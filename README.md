## 🚀🚀 ReactJs | Chrome Extension Starter Kit 🚀🚀
## Features

- ⚛️ ReactJs (v18)
- 🌱 TailwindCSS (v3)
- ⛑ TypeScript (v4)
- 📖 Webpack
- 📏 ESLint — Find and fix problems in your code

### Development

To start the project locally, run:

```bash
yarn && yarn watch
```

## Documentation

### Scripts
